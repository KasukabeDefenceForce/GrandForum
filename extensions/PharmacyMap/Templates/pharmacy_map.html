Using the Community Program Library

Hover your mouse over a community resource to see a description. The + symbol means the resource category expands further. Click on the resource to see sub-categories. When you can no longer expand, the map will show you specific locations for that resource within KFL&A. Contact the program or service for more information, or to sign up. If you want to go back to the main library page, click the blue link on the top left.
