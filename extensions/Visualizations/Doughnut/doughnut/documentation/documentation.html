Installation

Copy the following files:
  1. jquery.js
  2. raphael.js
  3. popup.js
  4. doughnut.js
After the files are copied, a new instance of a doughnut chart can be created by making the following call: 
        $("#holder").doughnut(data);
Where "#holder" is the div tag that the chart will reside. data is either a url to a json file, or the javascript object itself.

Doughnut Properties

'data_type_singular'
string
required		Used to display beside the value in the popup text when the value = 1
'data_type_plural'
string
required		Used to display beside the value in the popup text when the value != 1
'sort'
string
Whether to sort the values or not. Options are "desc", "asc", "none" (default is "none)
'limit'
integer
The number of arcs to display per ring. If there are more than the specified limit, then the remaing entries are grouped into "Others" (default is no limit)
'width'
integer
The width of the drawing area (Default is the width of the container)
'height'
integer
The height of the drawing area (Default is the height of the container)
'legend'
Array
required		The legend used for the chart (see below)
'levels'
Array
required		The height of the drawing area (see below)

Legend Properties

'name'
string
required		The name of the data series
'color'
string
required		The color of the data series

Levels Properties

'labels'
Array<string>
required		The labels Array for the data series
'values'
Array<integer>
required		The values Array of the data series
